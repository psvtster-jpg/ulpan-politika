# Инструкция для редактора

## Как править партии

Откройте `parties-data.js`. Каждая партия — отдельный блок внутри фигурных скобок:

```js
{
  id: "likud",
  name: "Ликуд",
  leader: "Биньямин Нетаньяху",
  camp: "Правый лагерь",
  short: "Короткое описание.",
  explain: "Основное объяснение простыми словами."
}
```

Правьте текст только внутри кавычек. После каждой строки, кроме последней в блоке, должна оставаться запятая.

## Как добавить новую партию

1. Скопируйте любой существующий блок партии.
2. Вставьте его перед последней закрывающей скобкой `];`.
3. Замените `id`, `name`, `leader`, `camp`, `tags`, `topic`, `short`, `explain`, `audience`, `coalition`, `note`.
4. Проверьте страницу в браузере.

## Какие поля важны для продвижения

- `name` — должно совпадать с тем, как люди ищут партию по-русски.
- `hebrew` — помогает читателю сверить название на иврите.
- `short` — короткая фраза для таблицы.
- `explain` — главный текст карточки, пишем просто и нейтрально.
- `note` — редакционная пометка: что проверить перед публикацией.

## Редакторское правило

Не писать агитационно. Формулировки должны объяснять, а не убеждать голосовать за кого-то.

## Как обновлять “Главное за 30 секунд”

Откройте `current-news-data.js` и замените три строки в `items` на актуальные пункты из Telegram Ульпан Политики: `https://t.me/ulpan_politica`.

```js
items: [
  "Первый актуальный пункт.",
  "Второй актуальный пункт.",
  "Третий актуальный пункт."
]
```

Правило: это не должны быть случайные новости дня. Берите только то, что помогает понять предвыборный контекст и партии.
