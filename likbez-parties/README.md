# Страница предвыборного цикла по партиям Израиля

Это статический прототип страницы для Ульпан Политики. Его можно открыть локально как HTML, выложить на любой статический хостинг или позже перенести в Next.js, Astro, WordPress или CMS.

Перед продолжением работы сначала откройте `PROJECT_STATUS.md`: там зафиксированы текущие решения, что уже проверяли по публикации, и какие следующие шаги.

## Где что править

- `index.html` — структура страницы, SEO-теги, тексты крупных блоков, ссылки Telegram/Facebook.
- `current-news-data.js` — блок “Главное за 30 секунд”, который надо обновлять по текущим новостям из Telegram.
- `parties-data.js` — список партий, лидеры, объяснения, фильтры и короткие описания.
- `styles.css` — дизайн: цвета, отступы, сетка, мобильная адаптация.
- `app.js` — вывод таблицы, карточек и фильтрация.
- `og-image.png` — картинка предпросмотра для Telegram, Facebook и поисковой выдачи.
- `robots.txt` и `sitemap.xml` — файлы для индексации после замены домена.
- `EDITOR_GUIDE.md` — инструкция для сотрудников, которые будут править страницу.
- `PROJECT_STATUS.md` — журнал текущего состояния проекта и публикации.

## Что заменить перед публикацией

1. В `index.html`, `robots.txt` и `sitemap.xml` заменить `https://example.com/likbez/israel-parties` на реальный URL.
2. Проверить ссылки соцсетей: Telegram `https://t.me/ulpan_politica`, Facebook `https://www.facebook.com/ulpan.politica/`.
3. Если страница будет лежать не в корне, проверить путь к `og-image.png`.
4. Пройти фактчек `parties-data.js`: лидеры, названия, союзы, коалиционный статус, актуальные новые партии.
5. Перед публикацией обновить `current-news-data.js` по последним постам Telegram.
6. После публикации добавить страницу в `sitemap.xml` и подключить домен в Google Search Console.

## Быстрая публикация

Самый простой вариант — загрузить папку `likbez-parties` на Netlify Drop, Vercel или GitHub Pages. Для ручной передачи можно использовать архив `ulpan-politika-likbez-parties-site.zip`, если он создан командой из корня проекта:

```bash
zip -r ulpan-politika-likbez-parties-site.zip likbez-parties
```

После публикации проверьте:

- страница открывается по публичному HTTPS-адресу;
- в коде нет `example.com`;
- Telegram/Facebook показывают правильный предпросмотр;
- Search Console видит страницу и sitemap.

## Как расширять до новостного сайта

Рекомендуемая будущая структура:

- `/likbez/israel-parties` — эта страница-справочник.
- `/likbez/knesset` — как работает Кнессет.
- `/likbez/coalition` — как собирается коалиция.
- `/parties/likud`, `/parties/yesh-atid` — отдельные страницы партий.
- `/news/...` — новости.
- `/analysis/...` — аналитика.
- `/interview/...` — интервью.
- `/authors/...` — авторы и редакция.

Для Google и дальнейшего продвижения держите видимую дату обновления, авторство/редакцию, источники и отдельные страницы под частые поисковые запросы.
